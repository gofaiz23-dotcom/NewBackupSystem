# Backup System Frontend

React frontend for Backup System with component-based architecture.

## Installation

```bash
npm install
```

## Environment Setup

Create a `.env` file:
```
VITE_API_BASE_URL=http://localhost:5000
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Features

- ✅ Collapsible sidebar
- ✅ Settings management
- ✅ Table view with all data
- ✅ Add single/bulk settings
- ✅ Edit settings
- ✅ Delete settings
- ✅ Date filtering
- ✅ Responsive design
